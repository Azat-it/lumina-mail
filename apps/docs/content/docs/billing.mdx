## Credit System

Each email sent uses one credit:

- 1 credit = 1 email sent
- Credits never expire
- Buy credits in bundles

## Purchasing Credits

1. Go to Credits section on left sidebar

<ImageZoom src="/credit-selection.png" width="1200" height="1200" alt="Credit selection" />

2. Choose a credit package
3. Complete payment
4. Credits are added instantly

## Credit Usage

Credits are deducted when:

- Sending campaigns
- Sending test emails
- Automated emails
